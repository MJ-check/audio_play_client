# Audio Play Client

## 开放音乐平台PC端项目

&emsp;&emsp;PC端项目、前端项目。使用 nodejs + React 搭建，目前版本为 V1.1 ，后续会根据需求持续更新。

## 项目部署

1. 克隆项目到本地

   ```powershell
   git clone https://github.com/MJ-check/audio_play_client
   ```

2. 项目部署

   ```powershell
   npm install
   // 或者使用 cnpm
   cnpm install
   ```

3. 项目运行

   ```powershell
   npm run start
   ```

4. 项目打包

   ```powershell
   npm run build
   ```
